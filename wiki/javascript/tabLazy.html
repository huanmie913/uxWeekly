调用说明

一:Jquery版

  1. 1.先加载JQ库，再加载jQuery.tabLazy.js( jq )
  2. 2.参数说明:

    1) _tabHandle : tab切换

    2)_curClass : 当前选中类名

    3)_bdHandle : tab切换内容父容器

    4)_conHandle : 内容层class

    5)_flag : 是否加载过的标记

    6)_eventType : 事件对象

    7)_num :初始化第几个

    8)_tabCon : 内容层Id

    9)_callback: 回调函数

  3. 3.调用方式: TabLoad({ _tabHandle : $("#tab_ul li"), _curClass : "current", _bdHandle : $('#tab_bd'), _conHandle : ".m_tcon", _flag : "tflag", _eventType : "click", _num :0, _tabCon : "tabslide_", _callback: function(i){ tab_one(i); } })

效果预览:tab+延迟加载+左右滚动

二:原生js版

  1. 1.加载tabLazy.js( 原生js )
  2. 2.参数说明:

    1) hid : tab切换

    2)htag : 作用于标签

    3)currentClass : 当前选中样式

    4)bid : 内容层父框

    5)bnode : 内容框标签

    6)bclass : 内容框类名

    7)dPro :自定义属性

    8)num : 默认序号

  3. 3.调用方式: TabLazy({hid:"tab_type",bid:"tab_game",num:0}); })

效果预览:tab+延迟加载
